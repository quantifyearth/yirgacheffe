# Core

To get started with yirgacheffe you can import data using the following core methods.

::: yirgacheffe
    handler: python
    options:
        members:
            - read_raster
            - read_raster_like
            - read_shape
            - read_shape_like
            - read_rasters
            - constant
            - from_array
            - area_raster
            - to_geotiff
        show_root_heading: false
        show_source: false
